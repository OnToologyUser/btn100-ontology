Apartaderoc back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/Apartadero

Son estaciones de poco tráfico de viajeros y cuyo objetivo fundamental es la regulación del tráfico ferroviario, posibilitando la realización de cruces de trenes, adelantamientos, etc.
has super-classes
Enclave Ferroviario c

Apeaderoc back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/Apeadero

En los ferrocarriles, sitio de la vía preparado para el servicio público, pero sin apartadero ni los demás accesorios de una estación.
has super-classes
Enclave Ferroviario c

Autoviac back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/Autovia

Carretera con calzadas separadas para los dos sentidos de la circulación, cuyas entradas y salidas no se someten a las exigencias de seguridad de las autopistas.
has super-classes
((calzada op value doble) or (calzada op value sentido unico)) and ((estado de uso op value en construccion) or (estado de uso op value en uso))

Callec back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/Calle

Vía que se encuentra localizada dentro de un núcleo de población o Diseminado superficial, y que ayuda a identificar la estructura de manzanas de dicho núcleo/diseminado.

Cargaderoc back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/Cargadero

Instalación ferroviaria que permite el acceso de mercancías desde los trenes a los barcos. Se distingue de una estación en que no es parte de la gestión de la circulación, ni tiene que disponer de desvíos y señales.
has super-classes
Enclave Ferroviario c

Carretera Autonómicac back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/CarreteraAutonomica

Tramo de carretera convencional de competencia autonómica o de cualquier otra administración inferior a ésta.
has super-classes
carretera
((calzada op value convencional) or (calzada op value sentido unico)) and ((competencia de vía op value autonomica) or (competencia de vía op value local) or (competencia de vía op value provincial))
is in domain of
categoría vial op

Carretera Nacionalc back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/CarreteraNacional

Vía de comunicación dedicada al tráfico rodado, cuya competencia corresponde a la Administración Central del Estado, clasificada como carretera de primer orden por el Ministerio de Fomento (Red de Interés General del Estado).
has super-classes
carretera
(calzada op value convencional) or (calzada op value sentido unico)

Suministroc back to ToC or Class ToC

IRI: http://geo.linkeddata.es/ontology/Suministro

Son estaciones que aparte de regular el tráfico, tienen como misión principal la prestación de servicios de mercancías. Disponen de todas las instalaciones necesarias para la recepción, clasificación y formación, y expedición de los trenes de mercancías convencionales, que circulan entre ellas y/o a otros destinos nacionales e internacionales.
has super-classes
